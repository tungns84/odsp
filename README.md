# Open Data Integration Platform

A centralized web platform to integrate, query, and share data from multiple dynamic sources.

## Project Structure

*   **[backend/](backend/)**: Spring Boot Application (Java 21)
*   **[frontend/](frontend/)**: React Application (Vite + TypeScript)
*   **[docs/](docs/)**: Project Documentation

## Getting Started

### Backend
```bash
cd backend
mvn clean install
mvn spring-boot:run
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```
